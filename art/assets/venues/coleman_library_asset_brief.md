# Asset Brief — Samuel H. Coleman Memorial Library

## Asset Goal
Create a reusable main-library asset for study, research, and reflective academic comic scenes.

## Reference Notes
- Official name: **Samuel H. Coleman Memorial Library**
- Main FAMU library
- Address shown on official library pages: **525 Orr Dr., Tallahassee, FL 32307**
- Built in **1948**, renovated in **1972**, expanded in **1990** and **2004**
- Includes study rooms, student lounge/cafe, graduate/faculty study areas, and information literacy classroom
