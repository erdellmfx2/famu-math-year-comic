# Asset Brief — Al Lawson Center

## Asset Goal
Create a reusable indoor basketball arena asset for FAMU sports and campus-life scenes.

## Asset Type
Background plate / environment reference

## Visual Identity
- indoor hardwood basketball arena
- modern collegiate seating bowl
- bright indoor lighting
- FAMU green and orange atmosphere
- energetic but clean environment suitable for comic staging

## Best Reusable Views
1. Wide court establishing view
2. Midcourt sideline view usable for game or conversation scenes
3. Lower-bowl crowd/background angle

## Prompt Guidance
Modern animated comic style, clean semi-cartoon linework, bright indoor sports lighting, Florida A&M basketball arena atmosphere, hardwood court, collegiate seating bowl, green and orange accents, readable environment, G-rated and social-media comic friendly.

## Reuse Value
High — can support:
- basketball game scenes
- pep rally / event scenes
- student-athlete or campus culture moments
- recruiting visuals

## Suggested Initial Generation Count
- 2 to 3 high-quality base images
