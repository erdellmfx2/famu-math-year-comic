# Reusable Asset Library

This folder defines the compute-saving assets for the FAMU Math Year Comic.

## Goal
Reduce image-generation cost by creating reusable assets instead of generating every comic panel from scratch.

## Asset Types
- `characters/` — repeatable character reference packs, pose packs, expression sheets
- `backgrounds/` — recurring campus and venue environments
- `compositions/` — two-character conversation setups and other repeated scene layouts
- `props/` — reusable objects and academic items
- `venues/` — athletics and campus landmark reference briefs

## Production Rule
Generate the highest-reuse assets first. Reuse them in panel layouts before asking the image model for a brand-new scene.
